# 📊 Attendance Tracker (Streamlit App)

A simple web app built using Streamlit to calculate aggregate attendance percentage.

## 🚀 Features

- Enter number of classes attended
- Automatically calculates attendance percentage
- Displays warning if attendance is below 75%
- Clean and simple UI

## 🛠 Tech Stack

- Python
- Streamlit

## ▶️ How to Run Locally

1. Clone the repository:

git clone https://github.com/NikhilBhalothia/attendance-tracker-streamlit.git

2. Go inside the project folder:

cd attendance-tracker-streamlit

3. Install dependencies:

pip install -r requirements.txt

4. Run the app:

python -m streamlit run app.py
